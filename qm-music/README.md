## 项目概述

[qm-music](https://github.com/chenqimiao/qm-music) 是 chenqimiao 开发的开源私人音乐服务项目，使用 Java 构建，支持 Subsonic 协议。

## 核心功能

项目提供 Docker 容器化部署，轻量级设计仅需约 150MB 内存，支持音乐文件托管、转码和流媒体播放。 关键特性包括自动音乐库刷新、低资源消耗以及与 feishin 或 StreamMusic 等客户端兼容构建私人音乐库。

## 快速部署指南

访问 http://localhost:6688 开始使用，支持 Windows、macOS、Linux 和 NAS。 Docker Compose 方式同样简单，适合生产环境。

## 资源与社区

官方 Docker 镜像：qmmusic/qm-music:latest。项目强调隐私和自托管，适合音乐爱好者和自建服务用户。
